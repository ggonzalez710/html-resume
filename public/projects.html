Here's a list of my best projects!

  • Visual Basic Programming: Calculator

    A computer application with a graphical user interface that's a calculator capable of making basic mathematical calculations.

  • Python Game Development: Pac-Man

    I programmed the classic arcade game "Pac-Man" from scratch with graphics.py, an object-oriented graphics library. For my version of the game I created my own movement algorithm so that the ghosts that chase Pac-Man move randomly across the map.

  • Python Game Development: Solitaire

    This was a group project that consisted of programming the game Solitaire with only one library (graphics.py). It was designed to be virtually impossible to complete due to the graphics.py library's limitations, yet my group reached a 90% completion state, the highest of the class.

  • Visual Basic Programming: Resistor Calculator

    Programmed a computer application with a graphical user interface that accepts the ohm value of a resistor as input and then displays the resistor's appropriate color code; the program also allows the user to choose the resistor's color bands to then use that color code to calculate the resistance in ohms.
